UNDERSTANDING KEY METRICS
═════════════════════════════════════════════════════════════════════════════

SOCIAL UNREST INDEX (0-1)
├─ 0.0-0.3   → Stable society, no major issues
├─ 0.3-0.6   → Growing discontent, protests emerging
├─ 0.6-0.8   → Crisis conditions, high migration
└─ 0.8-1.0   → System collapse risk

UNEMPLOYMENT RATE
├─ <5%       → Healthy labor market
├─ 5-8%      → Normal/acceptable range
├─ 8-12%     → Recession conditions
└─ >12%      → Depression/crisis level

INFLATION RATE
├─ 1-3%      → Target range (healthy)
├─ 3-5%      → Elevated but manageable
└─ >5%       → Problematic price growth

WEALTH INEQUALITY (Gini Coefficient)
├─ <0.3      → Very equal distribution
├─ 0.3-0.4   → Moderate inequality
└─ >0.4      → High inequality (socially risky)

CITIZEN STRESS
├─ <0.4      → Content population
├─ 0.4-0.7   → Moderate anxiety
└─ >0.7      → Crisis stress levels

CITIZEN SATISFACTION
├─ >0.7      → Most citizens are happy
├─ 0.4-0.7   → Mixed satisfaction
└─ <0.4      → Widespread unhappiness

   └─ BEHAVIORAL_CONFIG → Thresholds

COMMON QUESTIONS
═════════════════════════════════════════════════════════════════════════════

Q: How do I run a 20-year simulation?
A: sim.run_simulation(timesteps=240)

Q: How do I test multiple policies together?
A: sim.policy_engine.set_policy("income_tax_rate", 0.35)
   sim.policy_engine.set_policy("universal_basic_income", 1000)
   sim.run_simulation(timesteps=120)

Q: Why is social unrest so high?
A: Check: unemployment rate, inequality, and inflation together
   Usually caused by high unemployment + high inflation + inequality

Q: What if I want different agent behavior?
A: Modify Citizen.py methods like _make_consumption_decision()

Q: Can I see what individual citizens are doing?
A: Yes! Use sim.get_citizen_data() to get all agent states

Q: How do I compare policies systematically?
A: Use Experimentation module:
   suite = ExperimentBuilder.create_scenario_suite()
   suite.run_all()
   suite.compare_results()

Q: Can this predict real economies?
A: No, this is a simplified model for exploring policy relationships
   Use for intuition, not precise forecasting

Q: What's the simulation speed?
A: 1000 agents × 120 months ≈ 10-30 seconds on modern hardware

TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Issue: ImportError when running scripts
→ Solution: pip install -r requirements.txt

Issue: Dashboard doesn't load
→ Solution: streamlit run Dashboard.py (run from correct directory)

Issue: Simulation crashes with large population
→ Solution: Reduce population size or increase available RAM

Issue: Results seem unrealistic
→ Solution: Check calibration settings in Config.py
           Compare with RealWorldCalibration values

Issue: Policy changes not taking effect
→ Solution: Make sure to run sim.run_simulation() after policy change
